🎞️ 영상처리 프로그래밍
📚 교과목 개요
본 과목은 스마트폰, CCTV, 블랙박스 등 다양한 영상 장치로부터 수집되는 영상 데이터를 기반으로,
의미 있는 정보를 추출하고 이를 시각적으로 분석하는 방법을 학습합니다.

디지털 영상 처리의 기초 이론부터 고급 영상 분석 기법, 그리고 기계학습 및 딥러닝 기반 영상 처리까지 전반적인 내용을 다룹니다.

매주 실습 과제 프로그래밍과 기말 프로젝트 제출이 포함되며,
Python 기반의 다양한 영상처리 라이브러리를 활용하여 구현 능력을 기르는 것을 목표로 합니다.

---

📌 주요 학습 내용
- 디지털 영상 샘플링 이론
- 주파수 영역(Frequency Domain) 이해
- 화소(Pixel) 단위 영상 처리
- 영상 내 영역 처리 (Region Processing)
- 영상의 기하학적 변환
- 주파수 영역 기반 영상 처리
- 영상의 형태학적 처리 (Morphology)
- 고전적 기계학습 기반 영상 분석
- 딥러닝 기반 영상 처리
